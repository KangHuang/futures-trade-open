# futures-trade

## 环境要求
需要 python3, pip3工具

* pip3 install requests

* 安装WebSocketclient。pip3 install ./websocket_client-0.44.0-py2.py3-none-any.whl


## 配置参数
| 名称| 说明|
|----|----|
|accessKey |用户accessKey，需要修改|
|secretKey | 用户secretKey，需要修改|
|host      | websocket地址，不需修改|
|protocol  | 协议，不需修改|
|path     |  路径，不需修改|
